## GitHub Pages 部署
1. 建立 GitHub 儲存庫
2. 上傳全部檔案到儲存庫根目錄
3. 設定 → Pages
4. 來源選 從分支部署
5. 分支選 main，資料夾選 /root
6. 儲存後等待 GitHub Pages 產生網址

網站首頁：`index.html`
